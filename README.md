# Flask_Project
Проект собственного сайта на базе Питона с фреймворком фласк + БД SQlite + SQLAlchemy + Flask-Login + FlaskWTF

Суть данного проекта заключалась в том, чтобы я смог научиться читать, изучать, понимать и разбираться в неизвестных для меня библиотеках, фреймворках и т.д. А также, одной из целью являлось соединение логики работы одной библиотеки с другиой, или с несколькими, чтобы обеспечить прочный фундамент, хорошую и эффективную работу для дальнейшего продвижения и развития данного проекта. 

Что включено в сам проект?

Проект построен на базе:

-SQLITE (Не копризная база данных, достаточно проста в использовании, создании, подключении, управлении, а также имеется очень хорошая и удобная библиотека для управления ею)

-SQLALCHEMY (Удобно управлять, искать, отображать нужные данные)

-FLASK WTF (Вещь, которая способна и защитить сам сайт от различных Хаккерских аттак, а также способная минимизировать рутину кода с добавлением одного и того же блока form. Инициализация происходит наряду с файлом __init__, )
